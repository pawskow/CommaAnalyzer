# CommaAnalyzer

Uruchomienie skrypytu:
```python run.py plik_wejsciowy.txt```

`plik_wejsciowy.txt` to argument opcjonalny - domyślnie otwierany jest plik `input.txt`.
Plik ten powinien zawierać tekst do przetworzenia (tekst może zawierać przecinki, zastaną one usunięte, a później oryginał wykrzystany do oceny jakości).

W trakcie działania skryptu, na wyjście zwracane są kolejne zdania wraz z oceną poprawności wstawiania przecinków.
Cały plik wynikowy znajdzie się w pliku `result/output.txt`

W katalogu `resources` znajdują się pliki z przykładowymi tekstami.
